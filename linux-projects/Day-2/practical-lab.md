# Practical Lab - Users and Permissions

## Objective

Practice creating users, managing groups, changing ownership, and modifying permissions.

---

## Tasks

### Task 1

Create a new user named `developer`.

---

### Task 2

Create a group named `devops`.

---

### Task 3

Add the `developer` user to the `devops` group.

---

### Task 4

Create a directory named `project`.

---

### Task 5

Create a file named `deploy.sh`.

---

### Task 6

Make the script executable.

---

### Task 7

Change ownership of the directory to the `developer` user.

---

### Task 8

Verify the permissions and ownership using:

```bash
ls -l
```

---

## Challenge

Configure a shared directory where only members of the `devops` group can create, modify, and execute files, while preventing access from all other users.
