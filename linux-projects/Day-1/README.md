# Day 01 - Linux Basics

## Objective

The goal of Day 1 is to understand the basic Linux file system, navigation commands, file permissions, environment variables, and the purpose of the `.bashrc` file. These concepts are fundamental for anyone preparing for Linux Administration, DevOps, Cloud Engineering, or Site Reliability Engineering (SRE) roles.

## Topics Covered

- Linux File System Overview
- Navigation Commands
- File Permissions
- Environment Variables
- The `.bashrc` File

## Why This Matters in DevOps

Linux is the most widely used operating system for servers and cloud environments. Most DevOps tools such as Docker, Kubernetes, Jenkins, Ansible, and Terraform are managed through the Linux command line. Understanding Linux basics is essential for troubleshooting, automation, and infrastructure management.

## Learning Outcome

After completing this module, you should be able to:

- Navigate through the Linux file system
- Create, copy, move, and delete files and directories
- Understand Linux file permissions
- Work with environment variables
- Customize your shell using `.bashrc`
